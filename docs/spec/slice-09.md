# slice-09 skillsheet-view — スキルシート閲覧

> 振る舞いの正本: `reference-mock/spec.md §3.5`（格納と履歴・ダウンロードは元 xlsx・プレビュー）・`§7`（スキルシート閲覧）。
> 依存: slice-08（生成済みシートが閲覧の入力）。フェーズ: Phase 2。画面: S6。
> REST 表現（パス/メソッド/コード）と画面 DOM は参照モックに無い＝🟥 `source: PM`。

## AC-1 自分の生成済みシートの一覧と生成履歴を表示できる

- source: reference-mock        # spec.md §3.5（格納と履歴）・§7（スキルシート閲覧）
- **Given** 生成済みシートが複数版あるスタッフ
- **When** スキルシート閲覧を開く
- **Then** 自分の生成済みシートが生成日時の新しい順に一覧表示され、再生成された各版が履歴として残っている

## AC-2 ダウンロードは元の xlsx を渡す

- source: reference-mock        # spec.md §3.5（ダウンロードは元の xlsx）
- **Given** 生成済みシート
- **When** ダウンロード操作を行う
- **Then** プレビュー用変換ではなく**元の xlsx** が署名付き URL 経由で取得できる

## AC-3 他人のシートは閲覧・ダウンロードできない

- source: PM        # ★参照モックは HTTP 認可コードを規定しない（overview §1 deny-by-default）
- 理由: 権限境界はバックエンドで強制する。自分のシートのみ閲覧可、他人は拒否。
- **Given** 他スタッフの生成済みシート
- **When** その ID でダウンロードまたはプレビューを要求する
- **Then** `403` が返る

## AC-4 閲覧 API の REST 表現

- source: PM        # ★参照モックは HTTP 表現を規定しない（overview §3）
- 理由: 一覧・ダウンロードのパス/メソッド/コードは新規設計。
- **Given** 認証済みスタッフ
- **When** `GET /skill-sheets`（一覧）／`GET /skill-sheets/:id/download`（xlsx）を呼ぶ
- **Then** `200`。存在しない ID は `404`、未認証は `401`

> **決定（PM・2026-07-15）**: プレビュー方式は **HTML 変換**に決定（ダウンロードは AC-2 のとおり元の xlsx）。
> source: PM。理由: MVP は軽量優先で HTML プレビューを採る（PDF 変換は採らない。overview §8.5 凍結項目8 の凍結解除）。
> プレビュー結果は HTML として表示されるため、下記 AC-5 で観測点を定める。

## AC-5 プレビューは HTML として画面に表示される

- source: PM        # ★プレビュー方式は参照モックに無い＝新規決定（overview §8.5 凍結項目8）
- 理由: プレビュー方式を HTML 変換に決定したため、プレビュー導線の結果を観測できる。ダウンロード（AC-2）とは別経路で、元 xlsx は渡さない。
- **Given** 生成済みシートと HTML プレビュー方式
- **When** 一覧のプレビュー導線を操作する
- **Then** そのシートの内容が**HTML プレビューとして画面に表示される**（PDF/画像ではなく HTML。ダウンロードは AC-2 の元 xlsx のまま）

## 画面要件

- 対象画面: S6 スキルシート閲覧（overview §2。対象: staff）
- golden 撮影: **一覧・履歴部分は可**。プレビュー領域は **HTML 変換に決定（PM）**したため、**HTML プレビューも撮影可**（もしくは HTML プレビュー表示の DOM アサーションで検証・ADR-0018）。
- **UI-AC（source: PM）**（参照モックに画面の DOM が無いため本当の設計）
  - 生成済みシートが生成日時の新しい順に一覧表示され、各行に版（生成日時）が識別できる。
  - 各行に xlsx ダウンロードの導線がある（AC-2 の元 xlsx を取得）。
  - プレビュー導線を操作すると、そのシートの内容が **HTML プレビューとして表示される**（PM 決定・AC-5）。

## 合成フィクスチャ（PM 所有）

**本番/実データ・PII・客先名を入れない**（憲法 §1-6）。

| フィールド | 階層 | 例 | 備考 |
|---|---|---|---|
| staff.name | L1 | `テスト 太郎` | 所有者（本人 PII） |
| sheet.id | L0 | `sheet_0002` | 一覧・DL の対象 |
| sheet.filename | L1 | `テスト 太郎_スキルシート_20260715.xlsx` | 一覧表示・DL ファイル名 |
| sheet.generated_at | L0 | `2026-07-15T09:00:00Z` | 履歴の並び順（新しい順） |
| sheet.file_url | L0 | `（署名付き URL・合成）` | 元 xlsx の取得元 |
| other_staff.sheet_id | L0 | `sheet_9001` | 他人のシート（403 検証用） |
