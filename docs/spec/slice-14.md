# slice-14 admin-console — 管理者コンソール/スタッフ一覧

> 振る舞いの正本: `reference-mock/spec.md §3.9`。REST/コードは `docs/design/overview.md §3`（`GET /admin/staff` manager）。依存: slice-06。
> **スコープ注意**: 本スライスは「担当グループのスタッフ一覧表示＋担当外の遮断」に絞る。
> - 報告状況は spec.md §3.9 の二値（報告済み／未報告）まで。**報告サイクル・5 ステータス（提出済み/遅延提出/未報告/欠勤/報告漏れ）は slice-15**。
> - 各行の操作（プレビュー・DL・再生成・全員分生成）と最新スキルシートの生成は **slice-09・slice-21**（本スライスは一覧の列として参照するのみ）。
> **決定（source: PM・2026-07-15）**: 権限は本バッチでは最小ロール（staff/manager/メンタルケア担当）＋グループ単位の担当に固定する。spec.md §3.9 の「担当グループのスタッフのみ表示」＝**グループ単位・バックエンド強制**で確定。**スタッフ×管理者の個人単位の担当関係、主担当/副担当の操作差・副担当の操作線引きは後続 slice-24 permission-model で扱う（PM 決定 2026-07-15）**。理由: 権限3軸の細部は多数の未決を含み answer key を持たないため、本バッチはグループ単位に絞って型を回す（overview §8.5）。既存 AC はそのまま活きる。

## AC-1 管理者は自分の担当グループのスタッフ一覧を取得できる

- source: reference-mock        # spec.md §3.9「スタッフ一覧：氏名・客先・最終報告・状況・最新スキルシート・操作」／REST は ★PM
- **Given** 担当グループを持つ管理者
- **When** `GET /admin/staff` を呼ぶ
- **Then** `200` が返り、担当グループのスタッフごとに 氏名・客先・最終報告日時・報告状況（報告済み／未報告）・最新スキルシートの有無 が返る

## AC-2 管理者は担当グループ外のスタッフを見られない（バックエンド強制）

- source: reference-mock        # spec.md §3.9「担当管理者には自分の担当グループのスタッフのみ表示」／強制点・遮断は ★PM
- 理由（強制点）: 参照モックは強制点を規定しない。可視範囲の絞り込みは表示ではなくバックエンド（service 層の認可境界）で強制する（overview §1・§7）。
- **Given** グループ G1 を担当する管理者と、グループ G2 のスタッフ
- **When** `GET /admin/staff` を呼ぶ
- **Then** 応答に G2 のスタッフは一切含まれない（画面での出し分けだけでなくバックエンドで除外）

## AC-3 グループ別タブで担当グループを切り替えて絞り込める

- source: reference-mock        # spec.md §3.9「タブはグループに対応し、担当管理者には自分の担当グループのスタッフのみ表示」
- **Given** 複数の担当グループを持つ管理者
- **When** 特定グループのタブ（グループ指定）で一覧を取得する
- **Then** そのグループのスタッフだけが返る。担当外のグループは選択肢に現れない

## AC-4 スタッフはこの一覧を取得できない（ロール境界）

- source: PM        # ★ロール境界の拒否コード 403 は overview §3 の新規決定
- 理由: 参照モックはコードを規定しない。`/admin/*` は manager 専用。staff ロールでの呼び出しは `403`。
- **Given** staff ロールのログインユーザー
- **When** `GET /admin/staff` を呼ぶ
- **Then** `403` が返る

## 画面要件

- 対象画面: S8 管理者コンソール/スタッフ一覧（対象: manager・PC 前提）
- golden 撮影: 可
- **UI-AC（source: PM）**
  - 担当グループがタブとして並び、選択したグループのスタッフが表形式で表示される。
  - 各行に 氏名・客先・最終報告・報告状況（報告済み／未報告のテキスト）・最新スキルシート・操作 の列が並ぶ（操作の実挙動は slice-09・21）。
  - 担当外グループのスタッフは表にもタブにも現れない。
